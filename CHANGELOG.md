# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-13

### Added
- Initial release of Arabic Sentiment Analysis project
- Jupyter notebook with complete analysis pipeline
- Dual preprocessing approach (standard vs advanced)
- Negation preservation for Arabic text
- Arabic light stemming using tashaphyne
- N-gram feature extraction (1-3 grams)
- Logistic Regression and Linear SVC models
- Side-by-side confusion matrix visualization
- Comprehensive documentation (README, QUICKSTART, TECHNICAL)
- Automated setup scripts for Linux/Mac and Windows
- Project structure with data, models, and results directories
- Requirements.txt with all dependencies
- MIT License
- Contributing guidelines
- .gitignore for Python projects

### Features
- **330K Arabic Reviews Dataset**: Large-scale sentiment analysis
- **Smart Preprocessing**: Preserves negations and uses stemming
- **Model Comparison**: Standard vs Advanced approaches
- **Visualization**: Confusion matrices and performance metrics
- **Reproducible**: Complete setup and execution instructions

### Documentation
- README.md: Project overview and setup
- QUICKSTART.md: 5-minute getting started guide
- TECHNICAL.md: Deep dive into methods and architecture
- CONTRIBUTING.md: Guidelines for contributors
- PROJECT_STRUCTURE.md: File organization reference
